# SmartAutoML

## Project Overview
SmartAutoML is an end-to-end machine learning automation system that:
- Handles preprocessing
- Performs model selection
- Compares multiple algorithms
- Selects the best model automatically

## Features
- Data preprocessing
- Classification and Regression support
- Model comparison
- Accuracy evaluation
- Download trained model

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   streamlit run app.py
